# Sobre
Essa pasta contém exercícios de python avançado voltados para estruturas avançadas e performance mental.

## Exercícios
lru: Implemente uma LRU Cache do zero (sem functools.lru_cache) usando dict + lista duplamente ligada.
ranking: Faça um “sistema de ranking” que mantém sempre os Top-K maiores valores com atualização rápida.
grafos: Crie uma função que detecta ciclos em grafos (DFS) e retorna o caminho do ciclo.
tokenizer: Faça um “parser” simples que transforma uma string em tokens (tipo mini interpretador).
couter: Dado um texto gigante, conte palavras usando:
    - dict normal
    - collections.Counter
    - versão otimizada com defaultdict
